Overview

My research and practical work in shell security focuses on understanding and exploiting various attack vectors, including:

  • Shellshock vulnerability
  • Cross-Site Request Forgery (CSRF)
  • Cross-Site Scripting (XSS)
  • Buffer Overflow attacks
  • Encryption and Public Key

These studies have been instrumental in uncovering system vulnerabilities and developing mitigation strategies to protect systems against such threats. All experimentation and research was completed with Oracle VirtualBox using a Linux environment.

Research Highlights

  • Shellshock Exploit: Demonstrated how environment variable injection can be used to compromise vulnerable Bash versions. Experimented with creating files and executing commands via crafted HTTP headers.
  • CSRF Protection: Investigated the role of same-site cookies and CSRF tokens in mitigating attacks that exploit browser trust mechanisms.
  • XSS Worms: Developed a self-propagating XSS worm to understand the impact of malicious JavaScript in web applications.
  • Buffer Overflow: Designed payloads to exploit memory vulnerabilities, emphasizing the importance of stack protection mechanisms.
